# UnitFramesImproved_Ascension
Modified version with my own change and some fix

New feature :
Added new slash command for health display.
- /ufi short      - Toggle short numeric (1k vs 1000)
- /ufi percent    - Toggle percent display (adds (100%))
- /ufi max        - Toggle / Max display (1000 / 1000)
- /ufi scale #    - Set frame scale (0.5 to 2.0) (unchanged)
- /ufi reset      - Reset all StatusText settings (unchanged)
